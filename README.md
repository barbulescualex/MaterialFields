### Adding MaterialFields To Your Project

---

Open your podfile and add MaterialFields under your target

```
target 'Your Project' do

  use_frameworks!
  
  pod 'MaterialFields'
  
end
```

Save then in your project directory run

``` pod install ```

### Getting Started
